# Market Frame of Reference — Step 8 Output

> **Purpose:** Captures the candidate market categories, the yes/no style evaluations, and the single recommended market frame; builds on Steps 1–7 (best-fit customers, competitive alternatives, unique attributes, value themes) and feeds Step 9 (trend layering) and Step 10 (sales story).

## Style evaluation — Head to Head

> Apply per candidate category (Step 4). Viable if the category is understood AND (you are the leader OR no leader exists yet AND you can compete on the established criteria).

| Question | Answer |
|---|---|
| Is the category well understood by buyers? | yes / no |
| Are you already the market leader in this category? | yes / no |
| Is no clear leader yet established in the category? | yes / no |
| Can you credibly demonstrate superiority on the currently accepted purchase criteria? | yes / no |
| Do you have the features, customer proof points, and brand to compete on the leader's terms? | yes / no |
| **Verdict** | viable / not viable |

## Style evaluation — Big Fish, Small Pond

> Apply to the leading candidate category (Step 5). Viable only if all six answers are yes.

| Question | Answer |
|---|---|
| Is the broader category well-defined and understood by buyers? | yes / no |
| Does a clear market leader exist? | yes / no |
| Is there an identifiable subsegment with a distinct, important unmet need? | yes / no |
| Can you construct a list of prospective buyers in the subsegment via research or data? | yes / no |
| Is the subsegment large enough to meet near-term revenue goals? | yes / no |
| Is your competitive advantage in this subsegment difficult for the leader to copy? | yes / no |
| **Verdict** | viable / not viable |

## Style evaluation — Create a New Game

> Only evaluate if no existing or adjacent category scored viable above (Step 6). Viable only if all five answers are yes. Highest-cost, highest-risk option — last resort.

| Question | Answer |
|---|---|
| Have you evaluated every plausible existing category and none showcases your true differentiators? | yes / no |
| Can you identify a massive enabling change (new technology, economic shift, regulation, behavior shift) that makes this category newly possible or necessary? | yes / no |
| Can you answer 'Why now?' with compelling external market forces (not just internal ambition)? | yes / no |
| Does your solution require both new category boundaries AND new purchase criteria simultaneously? | yes / no |
| Do you have a large, well-resourced company or patient investors willing to fund years of market education? | yes / no |
| **Verdict** | viable / not viable |

## Fill rules

- Generate **at least two** candidate categories via abductive reasoning before evaluating any style; include any adjacent market that passes the genuine-fit test (not borrowed buzz).
- Evaluate **all three styles**, not just the intuitively obvious one — record a verdict for each.
- Pick **exactly one** style as the recommended market frame; no hedging, no "two-of-these."
- The recommended style must pass **every** yes/no rule for that style — no partial passes. Head to Head: understood AND (leader OR no-leader-yet AND can-compete). Big Fish, Small Pond: all six yes. Create a New Game: all five yes.
- Record **why each non-chosen style was disqualified**, mapping each to the specific rule that eliminated it.
- **Create a New Game is the highest-cost / highest-risk option** — only choose it if the checklist genuinely supports it AND no existing or adjacent category fits. Default to an existing category; when a clear leader exists, prefer Big Fish, Small Pond over Head to Head.
- Name the **exact** market frame of reference — not a vague direction like "something in the data space."
- If Big Fish, Small Pond is chosen: the subsegment must be named, listable, and sized against near-term revenue goals.
- If Create a New Game is chosen: a 'Why now?' narrative grounded in external forces must be articulated.
